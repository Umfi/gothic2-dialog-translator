Translate

by Umfi using DeepL.
Information
  • Open your dialog script using the 'Choose File Button'. (Your skript won't be uploaded and stored on the server)

  • Click on the translate button. All the extracted text (AI_Output, description, Info_AddChoice) is going to be translated using the DeepL Translation Service. (Keep in mind that only 5000 chars can be translated per script, there for you maybe have to split the file.)

  • Copy the translated text from the DeepL site back into the 'English textarea'.

  • When you leave the 'English textarea' the translated script will be generated into the 'Translated Script textarea'.
